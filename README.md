#  Google-Style Real-Time Weather Dashboard

A modern, production-ready weather dashboard inspired by Google Weather.
Built using **FastAPI** for the backend and **React** for the frontend, the
application consumes **live data from OpenWeatherMap** to deliver accurate,
real-time weather insights with a clean, responsive user interface.

---

##  Overview

Weather applications must be fast, reliable, and intuitive.
This project demonstrates a **real-world full-stack architecture** for
consuming third-party APIs, processing data efficiently, and presenting
actionable insights to users in a polished UI.

The dashboard provides current conditions, forecasts, and visual trends,
while handling real-world issues such as API latency, missing data, and
network failures gracefully.

---

##  Key Features

- Real-time weather data using OpenWeatherMap API
- City-based search with dynamic updates
- Current conditions:
  - Temperature & feels-like
  - Humidity
  - Wind speed & direction
  - Atmospheric pressure
- Forecast insights:
  - Hourly forecast
  - 7-day outlook
  - Min / max temperature trends
- Interactive charts for weather trends
- Responsive, Google-inspired UI design
- Clean separation of frontend and backend
- Environment-based API key management
